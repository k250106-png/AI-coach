# AI Interview Coach

AI Interview Coach is a monorepo with two app folders:

- [AI-Interview-Coach-BWAI-Backend](AI-Interview-Coach-BWAI-Backend)
- [AI-Interview-Coach-BWAI-Frontend](AI-Interview-Coach-BWAI-Frontend)

## What’s inside

- Backend: Express/TypeScript API, interview flow, analytics, Firebase, STT/TTS, and chatbot services.
- Frontend: Next.js app with interview practice, dashboards, ATS tools, and multilingual UI.

## Local setup

Run the backend and frontend from their own folders using their package scripts.

## Repo note

If GitHub ever shows the app folders as collapsed links instead of normal directories, make sure the nested `.git` folders are not present inside the backend and frontend folders before committing.
